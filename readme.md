# Setup Web App React

## Front-end part of the web app using React, Express, and Bootstrap.


### Milestone 1: Basic setup and configuration
1) **Install dependencies and set up the project structure**

- react
- react-router
- axios
- bootstrap
- bootstrap-icons
- font-awesome inter font

### Milestone 2: Create basic components and pages
2) **Create a defaultLayout component to wrap all pages and include the navigation bar and footer.**
3) **Create 2 pages: Home and single film page.**

### Milestone 3: Connect front-end to back-end
4) **Configure the back-end app to receive requests from the front-end app, creating CORS configuration.**
5) **Use axios to fetch data from the back-end API and display it on the home page.**

### Milestone 4: 
6) **Use axios to fetch data for a single film and display it on the single film page.**

### Bonus
7) **Add styling to the components using Bootstrap and custom CSS.**
